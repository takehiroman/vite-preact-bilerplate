# Vite + Preact bilerplate

### Frontend
- [Preact](https://preactjs.com/)
- [Vite](https://vitejs.dev)
- [Biome](https://biomejs.dev)

### Command

|Script|	Description|
| ---- | ---- |
|dev	| 開発モードで実行|
|build	|プロジェクトのビルド|
|lint	|biomeでlintの実行|
|format	|biomeでファイルをフォーマットする|
|lint:style	|Stylelintを実行し、可能な箇所は修正する|
